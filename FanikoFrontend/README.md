# Faniko — Starter Frontend (Vite + React + Tailwind)

A fast starter for building the Faniko MVP landing + app shell.

## Quick start
```bash
npm install
npm run dev
```

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- Minimal, production-friendly structure


Faniko – Creator Platform (Local Dev README)

This repo contains the full Faniko project (startup creator platform). It includes a React + TypeScript + Vite frontend and a Node.js + Express backend.

—————————
FOLDER STRUCTURE
—————————
FanikoStartup/
├─ FanikoBackend/      ← Express API
└─ FanikoFrontend/     ← React + Vite app

—————————
NPM INSTALLS REQUIRED
—————————
Backend packages:
express
cors
multer
bcryptjs
nodemon (dev only)

Frontend packages:
react
react-dom
react-router-dom
vite
@vitejs/plugin-react
typescript
tailwindcss
postcss
autoprefixer
@types/react
@types/react-dom

—————————
INSTALL COMMANDS
—————————
Backend:
cd FanikoBackend
npm install

Frontend:
cd FanikoFrontend
npm install

—————————
RUNNING THE PROJECT
—————————
Start backend:
cd FanikoBackend
npm run dev   (or npm start)

Start frontend:
cd FanikoFrontend
npm run dev

Vite will give you the local URL (usually http://localhost:5173)

—————————
NOTES
—————————
• This repo is intended to show the project for employers
• Frontend and backend must run at the same time
• API handles authentication, profiles, posts, and media uploads
